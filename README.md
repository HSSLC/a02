## 操作方式
`方向鍵下`:0  
`方向鍵右`:1  
`F2`:截圖至`./SS/*.png`  
`Esc`:設定長度  
`R`:重設  
`H`:顯示/隱藏幫助畫面  
還有其他彩蛋自己找囉

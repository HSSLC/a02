# a02
某java開發的小遊戲：碼畜

## 來源
這個程式的想法與名稱不是我原創的，在這裡先說明。  
想法與名稱來自Thankcreate

## 使用方式
如果要直接下載遊玩的話請去[Release](https://github.com/HSSLC/a02/releases)下載最新版本  

## 操作方式
`方向鍵下`:0  
`方向鍵右`:1  
`F2`:截圖至`./SS/*.png`  
`Esc`:設定長度  
`R`:重設
`H`:顯示/隱藏幫助畫面  
還有其他彩蛋自己找囉

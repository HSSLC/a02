## 遊玩方式
畫面上看到1就按`方向鍵右`，看到0就按`方向鍵下`，最後會統計出時間與各項數據在右下角  
第一行是目前進度  
第二行是完成花費的毫秒數  
第三行是平均每秒能打幾位數  
第四行是平均每位數要花幾毫秒  
  
基本上要跟別人比較的話會用第三行的數據，越高越厲害

## 操作方式
`方向鍵下`:0  
`方向鍵右`:1  
`F2`:截圖至`./SS/*.png`  
`Esc`:設定長度  
`R`:重設  
`H`:顯示/隱藏幫助畫面  
還有其他彩蛋自己找囉
